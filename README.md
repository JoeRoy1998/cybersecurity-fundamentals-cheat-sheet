# Cybersecurity Fundamentals Cheat Sheet

## Overview
This project is a beginner-friendly cybersecurity reference created to demonstrate foundational security knowledge. It covers common malware types, authentication methods, the CIA Triad, defensive security concepts, and the relationship between threats, vulnerabilities, and risk.

## Topics Covered
- Malware Types
- Authentication Methods
- CIA Triad
- Common Attacks
- Basic Network Security Concepts
- Defensive Security Concepts
- Threat vs Vulnerability vs Risk

## Purpose
This project was created as a learning exercise and portfolio piece to reinforce core cybersecurity concepts and improve technical communication skills.

## Skills Demonstrated
- Cybersecurity fundamentals
- Risk-based security thinking
- Defensive security concepts
- Clear technical documentation
- Security+ aligned knowledge
